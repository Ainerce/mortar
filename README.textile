
Don't take this too seriously yet. This is a template generation tool to make Open Web App templates.

To use:

git clone https://github.com/jlongster/mortarific.git
cd mortarific
npm install ejs
./bin/build app-stub my-app-stub

Blocks that you can override:

head.html
footer.html
js_global.js
js_requires.js
js_init.js

You can find these in @project/www/index.html@ and @project/www/js/app.js@.
